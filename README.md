# Maji Ndogo Water Services Analysis
Project Overview
This project involves a comprehensive analysis of the water crisis in Maji Ndogo. Using a dataset of over 60,000 records, I analyzed water source types, pollution levels, queue times, and infrastructure costs to provide actionable recommendations for improving water access.

Data Sources
The analysis is based on the following datasets:

water_source: Information on source types (wells, taps, rivers).

well_pollution: Chemical and biological contaminant data.

visits: Logistics data including queue times and visit frequency.

water_source_related_crime: Safety data regarding water collection.

infrastructure_cost: Budgeting information for improvements.

Key Insights
Queue Times: Identified specific towns where residents wait over 2 hours for water.

Pollution: Discovered significant biological contamination (E. coli) in rural wells.

Infrastructure: Calculated the total budget required to upgrade broken home taps to shared community taps.

Tools Used
SQL: Data cleaning and complex joins.

Power BI: Created an interactive dashboard to visualize the "Project Progress" and source distribution.

Excel/CSV: Initial data exploration.
